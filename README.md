# BlackJack

Original BlackJack game created with React and JavaScript only.

## Description

### Rules

- BlackJack pays 3 to 2
- Dealer stops at 16

### Game life

1. Player bet
2. Player draws two cards
3. Dealer draws two cards
4. Player Hit or Pass
5. Dealer draws if needed.
